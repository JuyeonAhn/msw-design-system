색상·아이콘·텍스트 3중 신호 원칙
색각 이상(적녹색맹 약 8%)을 포함한 모든 사용자에게 의미를 전달하려면, 상태를 색상 단독으로 표현하지 않습니다. 아이콘과 텍스트를 함께 사용하는 3중 신호 원칙을 적용합니다.

저장 완료 — 월드가 저장되었습니다.
오류 발생 — 업로드에 실패했습니다.
주의 — 변경사항이 있습니다.
색각 이상 사용자는 위 세 박스를 구분할 수 없습니다.

lightbulb MSW 서비스는 다양한 사용자를 만납니다. 접근성은 "장애인을 위한 기능"이 아닌 모든 사용자를 위한 품질 기준입니다. 접근성이 좋은 UI는 SEO, 유지보수성, 성능 모두 개선합니다.
prefers-reduced-motion
전정 기관 장애, 편두통 등으로 인해 애니메이션이 불편한 사용자를 위해 모션을 줄이는 미디어 쿼리를 지원합니다.
/* 모든 애니메이션/트랜지션 기본 정의 */ .btn, .chip, .card { transition: background var(--duration-base) var(--ease-emphasis), box-shadow var(--duration-base) var(--ease-emphasis); } /* 모션 감소 요청 시 — 트랜지션·애니메이션 최소화 */ @media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } /* 스켈레톤 shimmer 제거 → 정적 배경으로 대체 */ .skeleton { animation: none; background: var(--bg-subtle); } }
